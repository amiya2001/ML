## 🧠 What is Machine Learning?

Machine Learning is a subset of AI where:

> 📊 We feed data to an algorithm → it learns patterns → makes predictions

### 💡 Simple Explanation:
- No hardcoding rules  
- Model learns automatically from data  
- Improves with experience  

---

## 🔢 Types of Machine Learning

### 1️⃣ Supervised Learning
- Learns from **labeled data**
- Example:
  - House price prediction  
  - Spam detection  

---

### 2️⃣ Unsupervised Learning
- Works on **unlabeled data**
- Finds hidden patterns  

**Examples:**
- Customer segmentation  
- Clustering  

---

### 3️⃣ Reinforcement Learning
- Learns via **rewards & penalties**

**Examples:**
- Game AI  
- Robotics  

---

## 🌍 Applications of ML

Some real-world platforms using ML:

- 📱 Instagram → Feed ranking  
- 🎬 Netflix → Recommendations  
- 🛒 Amazon → Product suggestions  
- 🔍 Google → Search optimization  

---

## 🧩 Types of Learning

### 📦 Batch Learning
- Trained on entire dataset at once  
- Offline learning  

**Examples:**
- Fraud detection  
- Sales forecasting  

---

### 📊 Mini-Batch Learning
- Data divided into smaller chunks  
- Common in deep learning  

---

### ⚡ Online Learning
- Learns continuously  
- Updates model in real-time  

**Examples:**
- YouTube recommendations  
- Ad systems  

---

### 🔍 Instance-Based Learning
- Learns by comparing data  
- No explicit model  

**Examples:**
- KNN  
- Similarity search  

---

### 🧮 Model-Based Learning
- Learns a function/pattern  
- Generalizes to new data  

**Examples:**
- Linear Regression  
- Logistic Regression  

---
